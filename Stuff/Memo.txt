
	솔루션 : 프로젝트를 담는 틀

	프로젝트에 여러 코드 파일을 만들게 되는데, Main함수가 여러 개일 경우 모두 실행시켜버리는 문제가 있음
	이 때, 실행시키고 싶지 않은 코드 파일의 '빌드에서 제외' 속성의 값을 '예' 로 바꿔주면 해당 파일은 빌드되지 않음으로 Main 함수 역시 실행되지 않음.

	마찬가지로 솔루션에 여러 프로젝트를 만들게 되는데, '시작 프로젝트' 속성의 값을 원하는 프로젝트로 설정해둬야 해당 프로젝트에서 Main 함수를 찾아 실행함.
